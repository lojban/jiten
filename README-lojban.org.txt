This repo contains jiten the software.  We also run jiten as a service, available dict.lojban.org like so:

    $ dict -h dict.lojban.org klama
    17 definitions found
    
    From Lojban to German [jbo->de]:
    
            Word: {klama}
            Type: gismu
      Gloss Word: {bewegen}

    [snip lots more lines]

This is also what drives the main search page on https://jbovlaste.lojban.org/

The service management framework for that service lives at https://github.com/lojban/jbovlaste-containers
